# Извлечённые коэффициенты

Декодирование из hex-дампа `coeffs_final_v2.txt` (objdump на устройстве).

## DefaultAlgoCoefficients (0x770b7e0)

| Адрес | Float64 |
|-------|---------|
| 770b7e0 | -1.0 |
| 770b8a0 | **0.5** |
| 770b8a8 | **0.5946** (≈ 2^(-0.75)) |
| 770b8b0 | **0.7071** (= 1/√2) |
| 770b8b8 | **0.8409** (≈ 2^(1/6)) |

## Интерпретация как Kp, Ki, Kd

- **Kp** = 0.5
- **Ki** = 0.5946
- **Kd** = 0.7071

**Важно:** В tc-sync дефолты 0.1, 0.01, 0.001 — меньше на порядок. Значения 0.5/0.59/0.71 могут быть в других единицах или для иного масштаба. Тестировать осторожно.

## D-массив (0x770a430)

В `coeffs_final_v2.txt` данные по адресу 0x770a430 не извлечены. Для получения:

```bash
# На устройстве
objdump -s -j .noptrdata /usr/share/shiwatime/bin/shiwatime | grep -A 2 "770a4"
# или
gdb -batch -ex "x/3g 0x770a430" /usr/share/shiwatime/bin/shiwatime
```

## Использование в tc-sync

```yaml
servo:
  algorithm: pid
  kp: 0.5
  ki: 0.5946
  kd: 0.7071
  interval: 1s
```
